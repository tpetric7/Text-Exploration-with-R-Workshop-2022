Nekaj sto osebkov je rojilo po prostoru. Brezglavo so se naveličali. Nekaj časa so potrebovali. Potem so se spomnili. Rojili so kot Rimska cesta. Zagrizli so si v kožuhe, se trdno držali in brenčali okrog zelene veje. Nekaj časa so bivakirali. Pofotografirali so zeleno vejo. Izvolili so najlepši klobuk, zagnali svoje glasove in si sproducirali štarterja. Štarter je potegnil svetlečo, kromirano damsko pištolo, nameril natanko v zrak, najavili pripravljenost, gromko zarjul tik pred zdajci in elegantno poknil. Vsi so mu čestitali, pomrmrali najtoplejše, zastavili smeri in startali.
Med potovanjem so se presenečene dvojice embalirale, svojci pa so jih precejali med zobmi. Prvič so se sprli, ko je pošla pitna voda. Iznajdljivi so lovili deževnico, kasirali lepe denarce, da bi le ne bilo kakšne kuge ali lakote. Od časa do časa so se spravili v gruče in pod točko razno, prešteli prisotne. Štarterji so že od vsega začetka pogrešali, kakorkoli so že preštevali.
Kroniko dogajanj so slovesno načeli, ko je Verbičeva mala pripovedovala frkolinom o žehtečih problemih svoje Zijalke. Stari Verbič je obešenjaško preminil, frklja pa je odslej spremljala karavano v dražeči distanci, na večer se je šele približala taborišču.
Brezzobci so trudoma popustili opravke, zmajevali nad svetovno situacijo, bezali po grmovju in se pridušali. Bili so slabi časi za karavanskega kronista.
Smrkavci so staknili strupenega kuščarja in ga pobili s kamenjem. Matere so se zaradi tega hudo spraskale. Tisti dan so goloroki možje imeli mnogo opravka. Nekaj žensk je pobegnilo v črno noč. Obup je bil na višku.
Odkar je Pečanka imela videnje, so pošiljali krepke patrulje v bližnjo okolico. Poslej so bili tereni varni. Bogatinci so mirno spali, goli siromaki so imovito križarili krog voglov in prikrito pobirali. Katerekrati so kakšnemu dali po prstih, ženske so se pokrižale in pogrezajoče zavzdihnile.
Začele so krožiti govorice, da potujejo. Skrivnostno je tisti dan izginilo nekaj goloritcev. Najbolj so trpeli mulci, zaradi večerne ure. Patruljiranje so poostrili.
Brezzobci so se izkazali. Na razsipnih pojedinah so se najbolj nasesali. Tokrat so prireditve bile v slavo novorojenih. Bila je izredna žetev. Najbolj so se izkazale žene bogatinov. Čez karavano se je vlekla spokojnost. Kronisti so bili v zadregi.
Otorčad se je spet spraskala med seboj. Največ posla so imeli razvezanci. Nejevolja je prekipela. Pretepli so vse dninarje, ki so jih presenetili na njihovi bučni veselici. Ko so si opomogli od presenečenja, so poslali okrvavljene povoje vsem finim gospem. Postali so mirni in srenja je blagrovala njihovo spametovanje. Na samotne vrišče v oblačnih nočeh so se dokaj hitro privadili.
Skupnost si je oddahnila. Mučnih zapetljajev je bilo treba, da jih je zapustila učiteljica in se protestno pridružila Verbičevi deklini. Pečanovemu mlečnozobnežu so pa naložili nekaj toplih. Pečanko so tudi krepko potrepljali, ker je nagovoril starejšega Zelenčevega, da je med slovesno povorko učiteljici bezal s prstom po popku. Zaslužni so nekaj časa vrtali, odklej ta predrzna navada v njihovih vrstah, čez čas so opustili iskanje o izvoru pojava.
Najstarejši konjederec je na sestanku zaprosil za upokojitev, krepko je argumentiral svojo nejevoljo zavoljo psov, ki se bojda sami končujejo, ozmerjal za zaključek pasjo radost, križem pogledal in poudaril o elegičnih časih prisotnih dni, mlademu rodu pa položil na srce vso težo svoje kreativnosti in kulturi v odnosu da pravkar razloženega. V nadaljevanju je zvito mežikal in čakal na ugodno rešitev kompetentnih. Neprošeni so se živčno presedali, mleli z nogami in si podpirali z rokami očesne oboke. Rešitev konjederčeve rešitve so preložili v prihodnost, zaradi nenadejane prisotnosti Verbičeve dekline in učiteljice.
Srenja se je zavila v praznično žalovanje, ker so našli gnilo truplo starega štarterja, ki jim je povzročal težave pri preštevanju, kakorkoli so že šteli. Oddahnili so si, čeprav je bilo nelepo od njega, da jim je delal preglavice. Obdukcija je pokazala, da so starega štarterja kavsale tudi vrane ali kakšne druge ptice in zapečatili zadevo ad acta.
Tistega dne se je nekoliko premaknilo, ker so organizirali lepo praznovanje, da bi zatrli vsakdanje tegobe. Vse se je srečno končalo, razen nekaj malih izgredov, nevrednih besed, ki so jih povzročili neznani motitelji, katere so postavili na ključna mesta. Ključavničarji so imeli na praznovanju najbolje pripravljen program.
Mali Zaletelov je fotografiral sončni zahod, nakar je priredil razstavo na javnem prostoru. Staremu je sindikat nabil odtegljaje, vsi ostali dninarji pa so bili uradno osumljeni.
Starejšega Zelenčevega so opomnili, staremu pa zabičali, naj ostro pazi na sina, ker je pokvaril Pečanki videnje. Našli so jo prepoteno in na skrajnem koncu, ko je fantalin bežal. Uboga ženska trpi.
Skoraj bi prišlo do stare tragedije. Otročad je zopet staknila nekega kuščarja in ga pobila s kamenjem. Enotno mnenje se je hudo zamajalo. Po zaslugi zdrave pameti so obdržali ravnovesje. Ženske so se nekoliko unesle in niso ponovile starega greha.
Skoraj bi žalostno izginila Bogatajeva familija. Sina edinca so rešili v zadnjem trenutku, ko je klical na pomaganje. Verbičevi nasilnici so iskreno zagrozili, da jo bodo dokončno potolkli, če se še kaj takega zgodi. Od tistega trenutka se približa samo v zelo trdi temi.
Blagodejni mir so začeli kaliti parajoči vriski in zlovešče hihitanje, ki se je začelo pojavljati celo v zgodnjih večernih urah. Pojavile so se tudi neosnovane govorice o dobri letini. Vreme ni nič kazalo na izboljšanje. Zaskrbljenost se je naselila v vsako kolikor toliko zdravo hišo.
Staremu Zaletelu so zagotovili življenjski minimum na račun njegove invalidnosti, ko je reševal sina. Vdova si bo že še pomagala pri njenih še vedno obetavnih letih.
Zatrli so govorice o potovanjih. Trije dninarji so pripravili lep poslovilni proces. Obljubili so lepo predavanje z diapozitivi, ko se vrnejo. Nekateri so se škodoželjno posmehovali. Opomnjeni so bili in poučeni o škodljivosti izpodkopavanja morale in negotovitve osebnosti. Zaenkrat je zaleglo.
Stari Bogataj je energično preprečil novo katastrofo, ki je grozila familiji. Bogatajka ne sme več sama osebno nositi malice njihovim dninarjem. Stari je štiri pognal in najel novih pet. Petega so nastavili za hišne posle. Nekaj časa so brbljali o tem. Odpuščeni so šli v Ameriko.
Patrulije so podvojili. Pojavilo se je nekaj klatežev, ki imajo rovarjenja v planu. Najbrž so kakšni tuji elementi, ker jih ni bilo še mogoče identificirati.
Konjederec ni uspel dočakati svojega veselja. Poslovil se je ravno, ko je bila njegova ugodna rešitev na poti. Iz prostovoljnih prispevkov so priredili krasen pogreb. Vsa soseska je prišla gledat. Pasla nadlega postaja iz dneva v dan neznosnejša.
Nepričakovano je umrl kronist. Analitiki triumfirajo. Na vsesplošni veselici so se zaklali trije potepuhi. Nepopravljivo jo je skupil stari Bogataj. Verbičeva se je poboljšala. Zaletelov smrkavec je izgubil fotoaparat. O učiteljici ni ne duha ne sluha, Pečanko je vzela bridka smrt.
